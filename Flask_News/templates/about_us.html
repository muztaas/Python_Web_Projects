About Us


This is a web application developed using Python Flask by M.H.Ashraf.

It uses NEWSAPI with an API Key to bring latest news in a collected format which in grouped either by country or by a user specified keyword.

This application uses only the free version of NEWSAPI which has certain shortcomings like some delay in latest news, fixes languages and not covering all minor news.
